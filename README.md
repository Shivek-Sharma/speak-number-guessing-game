# Speak Number Guessing Game
A number guessing game where you speak your guess into the microphone using the speech recognition API.

# Project Specifications
• Display UI directing user to speak his/her guess into the microphone                                   
• Implement speech recognition to listen to mic                                                          
• Process user's guess and match with the correct number                                                 
• Let user know higher, lower, match, not a vaild number or not in range                                 
• User get 10 chances to guess the number                                                                
• Play again button to restart the game                                                                  
• Speech Recognition API is not compatible with Firefox and Brave Browser
